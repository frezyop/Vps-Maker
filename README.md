# 🚀 VPS-in-VPS Auto Creator

Create a new VPS inside your existing VPS with just one command.

✨ Features

- ⚡ One-command installation
- 🐧 Supports Ubuntu
- 📦 Automatically installs all required dependencies
- 💾 Creates a new isolated VPS/container
- 🖥️ Configures CPU, RAM, and Storage
- 🌐 Sets up networking automatically
- 🔄 Fast and beginner-friendly

📋 Requirements

- Ubuntu 22.04/24.04
- Root access
- Virtualization support

🚀 Installation
```bash
bash <(curl -s https://raw.githubusercontent.com/frezyop/Vps-Maker/main/vps.sh)
```

📖 Usage

Follow the on-screen instructions to create your VPS.

Example:

- CPU: 4 Cores
- RAM: 8 GB
- Storage: 100 GB

The script will automatically:

- Install dependencies
- Create the VPS/container
- Configure networking
- Start the VPS
